학습내용

1. 객체지향 언어 기반의 상속과 다형성 학습
	필요성 : 이미 구현된 프로젝트들 분석 및 수정
				API활용을 위한 필수 지식
2. 상속이란?
	***필요성: 코드 개발 속도 향상, 재사용, 확장 용이, 비용 절감
	1. 현실 세계와 동일 즉, 이미 존재하는 class의 멤버 변수와 method를 새로운 class에서 상속받아 재사용(복붙 아님)

	2. 다형성
		- 상속이 전제조건
		- 조상, 부모 타입의 변수에 자식, 자손 객체 대입 가능
		- 단 상위타입은 절대 상속시킨 멤버 변수와 메소드를 제외하곤 호출 불가
		- 자식만의 새로운 멤버 변수, 메소드 호출을 위한 해결책
		  : 객체 타입간의 형변환
		  자식타입 변수 = (자식타입)부모타입 변수;
3. 용어
	1. 부모
			- super/parent
	2. 자식
			- sub/child
			
4. 자바 특징
	1. java.lang.Object
		-모든 클래스의 절대적인 최상위 root 클래스
	2. 문법
		1. class 클래스명 extends 부모클래스명{}
			- 직계 부모 클래스는 하나의 클래스만 가능
			
		2. 상속은 멤버변수와 메소드 두가지만 상속 가능
